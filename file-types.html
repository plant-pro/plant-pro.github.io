Типы файлов

PDF

Метка формата: .pdf

PDF (Portable Document Format) - это широко распространённый формат документов. На данном сайте в формате PDF представлена большая часть литературы. Чтение данного формата встроено во многие современные браузеры, но можно воспользоваться специализированными программами для чтения: LibreOffice, PDF24 Creator

DJVu

Метка формата: .djvu

DJVu - технология сжатия изображений с потерями, разработанная специально для хранения сканированных документов — книг, журналов, рукописей и прочее, где обилие формул, схем, рисунков и рукописных символов делает чрезвычайно трудоёмким их полноценное распознавание. Для чтения можно использовать DjVu Reader

CSV

Метка формата: .csv

CSV (Comma-Separated Values, значения разделённые запятыми) - текстовый формат, предназначенный для представления табличных данных. Строка таблицы соответствует строке текста, которая содержит одно или несколько полей, разделенных запятыми.

TSV

Метка формата: .tsv

TSV (Tab-Separated Values, значения разделённые табуляцией) - текстовый формат, предназначенный для представления табличных данных. Строка таблицы соответствует строке текста, которая содержит одно или несколько полей, разделенных знаком табуляции.
