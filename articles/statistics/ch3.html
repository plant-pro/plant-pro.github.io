Статистика. Часть 2. Меры центральной тенденции

Вспомним пример из прошлой части: 30 студентов набрали на школьном тесте следующее количество баллов:

86 80 25 77 73 76 100 90 69 93
90 83 70 73 73 70 90 83 71 95
40 58 68 69 100 78 87 97 92 74

Как мы можем описать эти данные? Один из вариантов мы рассмотрели - изобразить графически, однако, этот способ не может дать нам много информации. Другой вариант - рассчитать меры центральной тенденции (measures of central location), которые позволяют ответить на вопрос "где находится центр данных?".

1. Среднее


Среднее - это мера центральной тенденции, хорошо знакомая всем.

Определение
Среднее выборки (sample mean) - это число, определяемое по формуле:
` bar x = (sum_(i=1)^n x_i)/n `
Где:
`n` - размер выборки
`sum_(i=1)^n x_i` - сумма всех наблюдений выборки

Определение
Среднее генеральной совокупности (population mean) - это число, определяемое по формуле:
` mu = (sum_(i=1)^N x_i)/N `
Где:
`N` - размер генеральной совокупности
`sum_(i=1)^N x_i` - сумма всех наблюдений генеральной совокупности

Например:

Пример
`x = { 2, -1, 0, 2 }`
`bar x = (sum_(i=1)^4 x_i)/4 = (2 + (-1) + 0 + 2)/4 = 3/4 = 0.75`

Для нашего примера мы таже можем рассчитать среднее:

Пример
`bar x = (sum_(i=1)^30 x_i)/4 = 2330/30 = 77.6`
Найти среднее по шагам:
  • 1. Подсчитайте сумму всех элементов вашей выборки
  • 2. Подсчитайте количество элементов вашей выборки
  • 3. Разделите сумму на количество элементов

2. Медиана


У среднего обычно существуют некоторые проблемы. Например, найдём среднее этой выборки:

Пример
`x = { 24.8, 22.8, 24.6, 192.5, 25.2, 18.5, 23.7 }`
`bar x = 47.4`

Очевидно, что для этой выборки среднее слишком велико из-за того, что там имеется одно значение (`192.5`), которое резко отличается от всех остальных. Такие значения называются выбросами.

Определение
Выброс (outlier) - это значение, которое сильно отличается от основного набора данных и которое может исказить результаты исследования.

В таком случае более подходящей мерой центральной тенденции будет медиана.

Определение
Медиана выборки (sample median) - это значение, которое лежит ровно в центре упорядоченной выборки и обозначаемое как `tilde x`

Обратите внимание на слово "упорядоченной" - перед поиском медианы вы должны расположить ваши значения в порядке возрастания. Найдём медиану примера выше:

Пример
`x = { 24.8, 22.8, 24.6, 192.5, 25.2, 18.5, 23.7 }`
Отсортируем выборку:
`x = { 18.5, 22.8, 23.7, 24.6, 24.8, 25.2, 192.5 }`
`tilde x = 24.6`

Итак, медиана в этом случае равна `24.6`, так как это число лежит ровно в середине и разделяет нашу выборку пополам. Но как нам найти медиану в примере с тестом? Ведь там 30 элементов и нет такого числа которое бы делило данные пополам! В таком случае медиану находят, складывая два средних числа и деля их пополам (конечно, выборка перед этим должна быть отсортирована).
Найти медиану по шагам:
  • 1. Расположите элементы вашей выборки в порядке возрастания
  • 2. Если размер выборки нечетный - медиана это элемент в середине
  • 3. Если размер выборки четный - сложите 2 элемента посередине и разделите на 2
